# 6.8 문제: 시계 맞추기(문제 ID: CLOCKSYNC, 난이도 중)

[문제 링크](https://www.algospot.com/judge/problem/read/CLOCKSYNC)

<br/>

## 내 풀이 

<br/>

- 문제 이해 

16개의 시계가 주어진다. 각 시계는 3, 6, 9, 12중 하나를 가리키고 있다. 

문제에서는 16개의 시계를 모두 12시를 가리키게 만들고 싶다. 시계와 연결된 스위치는 10개이고, 각 스위치는 여러 개의 시계와 연결되어 있다. 스위치를 누르면 연결된 시계들은 3시간씩 앞으로 움직인다. 

이때 모든 시계를 12시로 만들기 위해 최소한 스위치를 몇 번이나 눌러야 할지 구하는 것이 문제이다. 

<br/>

- 계획 

완전 탐색을 어떻게 적용해야 할지 잘 모르겠다...
